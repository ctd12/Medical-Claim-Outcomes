Stratified Sampling

Stratified sampling, or proportional random sampling, divides the population into a smaller subgroup that is representative of the population as a whole. The reason for using this sampling is that the overall population is very large, so performing initial modeling on a smaller subset, or strata, will save time. The strata will contain approximately 2.5% of all claims.

Code 
## Adding the stratified variable key

strata_claims <- claims %>%
  mutate(strata_key = paste0(week, month, year))

## Updating strata_key to a factor

strata_claims$strata_key <- as.factor(strata_claims$strata_key)

## Creating the stratified subset

strata_claims <- strata_claims %>%
  group_nest(strata_key) %>%
  mutate(samp_data = map(data, \(x) slice_sample(x, prop = .025, weight_by = strata_key))) %>%
  select(samp_data) %>%
  unnest(cols = everything())

## Remove the strata_key for the purpose of modeling

strata_claims <- strata_claims[,-249]

Strata Modeling

Five different modeling techniques will be used to classify the claims data. Each will be analyzed to compare accuracy and efficiency.

First, the data must be split into 80-20 training and testing sets. The true y values (claim status) will also be saved into a list for accuracy testing.

set.seed(179)
strata_rows = length(strata_claims$claim_status)
flag = sample(1:strata_rows, strata_rows*0.2, replace = FALSE)
strata_train = strata_claims[-flag,]
strata_test = strata_claims[flag,]

## True y values for the training and testing sets

true_strata_train <- strata_train$claim_status
